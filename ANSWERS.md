---

## 2. Stack & design choices

I used HTML, CSS, and Vanilla JavaScript because this project is lightweight and does not need a framework.

- Timer is centered and large so focus stays on time.
- Grid layout is used to separate timer and history clearly.
- Circular progress ring is used to visually show time progression.

---

## 3. Responsive & accessibility

- On 360px mobile screens, layout becomes single column and elements scale down.
- On 1440px screens, two-panel layout is used.

Accessibility:
- Buttons have clear labels.
- Good color contrast is maintained.

Skipped:
- Full ARIA screen reader support due to time limits.
